# HR - просто

MVP-платформа для управления проектными брифами и предложениями от фрилансеров.

## Возможности

- Авторизация с паролем
- Создание проектов с 6 типами шаблонов
- Публичные ссылки для получения предложений
- Админ-панель со статистикой и управлением пользователями
- AI-помощник (OpenAI gpt-4o-mini)

## Локальная разработка

```bash
# Установка зависимостей
npm install

# Применение схемы БД
npm run db:push

# Запуск в режиме разработки
npm run dev
```

Приложение будет доступно на http://localhost:5000

## Deploy to Timeweb

### Настройки деплоя

| Параметр | Значение |
|----------|----------|
| **Окружение** | Node.js |
| **Фреймворк** | Express |
| **Версия Node** | 20 или 22 |
| **Команда сборки** | `npm install && npm run build` |
| **Команда запуска** | `npm run start` |
| **Путь проверки состояния** | `/health` |

### Переменные окружения (обязательно)

| Переменная | Описание |
|------------|----------|
| `DATABASE_URL` | Строка подключения к PostgreSQL |
| `SESSION_SECRET` | Секретный ключ для сессий (длинная случайная строка) |
| `NODE_ENV` | `production` |

### Миграция базы данных

После настройки переменных окружения выполните **один раз** вручную:

```bash
npm run db:push
```

**Важно:** Не добавляйте `db:push` в команду сборки - это может вызвать ошибки, если БД недоступна во время сборки.

## Структура проекта

```
├── client/           # React фронтенд
│   └── src/
│       ├── components/   # UI компоненты
│       ├── hooks/        # React хуки
│       └── pages/        # Страницы
├── server/           # Express бэкенд
│   ├── routes.ts     # API маршруты
│   ├── storage.ts    # Работа с БД
│   └── ai.ts         # AI интеграция
├── shared/           # Общий код
│   └── schema.ts     # Схема БД (Drizzle)
└── dist/             # Собранное приложение
```

## Скрипты

| Команда | Описание |
|---------|----------|
| `npm run dev` | Запуск в режиме разработки |
| `npm run build` | Сборка для production |
| `npm run start` | Запуск production сервера |
| `npm run db:push` | Применить схему БД |
